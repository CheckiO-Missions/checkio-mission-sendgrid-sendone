Create a function that sends welcome email to a user. The function gets two arguments email and name of the user.

Subject of the email should be "Welcome" and body simply "Hi, {name}" ({name} should be replaced by users name)

You can send email FROM any email you want

In order to send email you should use sendgrid v3 API and registrate your API Key on SendGrid

Input: Two arguments. email and username

Output: None. 

    send_email('a.lyabah@checkio.org', 'oduvan')
    send_email('somebody@gmail.com', 'Some Body')

How it is used: It is important to welcome your user by email after one registered on your site.
